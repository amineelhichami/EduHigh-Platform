# EduHigh-Platform
EduHigh is a modern English learning platform designed for high school students, combining structured lessons, interactive exercises, and a clean user experience to make learning English simple, engaging, and effective.

## 🚀 Features
- Interactive English lessons
- Clean and modern UI
- Student-focused experience

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
